# NodeJS and mongoDB for Study Group Internship

## Prerequisite
1. NodeJS
    - Please install the Last Features of NodeJS (7.90)
        - Link download: [NodeJS download](https://nodejs.org/en/)
2. MongoDB
    - Please install the Last Version of MongoDB
